# MonFrench — Charte graphique (espace élève)

Esprit : « académique parisien » — la crédibilité Sorbonne/examinateur de Débonnaire, avec la chaleur de sa devise (optimisme, patience, encouragement). Minimal : rien de plus que ce dont l'élève a besoin.

## Typographie

- Titres / logotype : **Cormorant Garamond** (Google Fonts) — sérif classique, esprit universitaire.
- Interface : **Inter** — lisible, moderne, neutre.

## Logo

Logotype « Mon**French** » (« French » en or) surmonté d'une fleur de lys ⚜ dans un cercle doré fin. La fleur de lys reprend directement ton profil italki. Réutilisable seule comme favicon/icône.

## Image de page d'accueil

Pas de photo. La page de connexion EST la page d'accueil : fond bleu nuit avec un léger halo doré, le logo, la devise, et le formulaire. Zéro information exposée (pas de nom d'élèves, pas de liste de cours, pas de description). Si tu veux une touche visuelle plus tard : une illustration linéaire discrète (toits de Paris ou fenêtre haussmannienne) en filigrane doré à 8 % d'opacité derrière la carte de connexion — jamais une photo chargée.

## Textes de l'interface (exacts)

**Connexion**
- Devise sous le logo : *Votre espace de français, simplement.*
- Champs : `Identifiant` / `Mot de passe`
- Bouton : `Se connecter`
- Aide : `Identifiant oublié ? Contactez votre professeur.` (pas de reset par email → pas d'email exposé)

**Espace élève**
- Accueil : `Bonjour, {prénom} 👋` + `Vous avez {n} activités à faire.`
- Onglet 1 : `À faire` (avec compteur)
- Onglet 2 : `Envoyé au professeur` (avec compteur)
- Carte à faire : titre + `À rendre le {date}` + type/durée + bouton `Commencer`
- Carte envoyée : titre + `Envoyé le {date}` + `✓ Envoyé`
- Bouton : `Se déconnecter`

## Principes de confidentialité / minimalisme

1. Page publique = uniquement logo + devise + formulaire de connexion.
2. Aucune liste d'élèves, aucun nom de classe, aucune statistique visible avant connexion.
3. Après connexion : seulement les deux listes (à faire / envoyé). Pas de notes, pas de profil, pas de fil d'activité.
4. Prénom seul dans l'accueil (jamais nom complet).
5. Récupération de mot de passe via le professeur, pas d'email affiché.
