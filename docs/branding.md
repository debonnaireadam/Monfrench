# MonFrench brand guide

## Direction

Academic Parisian: credible, calm, encouraging, and deliberately minimal.

## Palette

- Sorbonne navy: `#1C2B4A`
- Deep navy: `#141F38`
- Cream: `#FAF6EE`
- White: `#FFFFFF`
- Gold: `#C9A227`
- Ink: `#22283A`
- Slate: `#6B7285`
- Sent status: `#2E7D5B` on `#EAF4EF`
- Due status: `#8A6D1B` on `#FBF3DC`

## Typography

- Display and logotype: Cormorant Garamond
- Interface: Inter

## Privacy principles

The public page exposes only the brand, tagline, and sign-in form. Student names, course lists, and activity data remain behind authentication. The student portal uses first names only and shows only assigned and submitted work.
